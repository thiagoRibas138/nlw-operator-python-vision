# NLW Operator - Trilha Python & Computer Vision 🚀

Projeto desenvolvido durante o NLW Operator da Rocketseat. A aplicação utiliza Python e bibliotecas de Visão Computacional para processamento de imagens/vídeos.

## 🛠 Tecnologias
- Python 3.x
- OpenCV / Mediapipe (ou as que você usou)
- Git & GitHub

## 📝 O que o projeto faz?
- [ ] Detecção de objetos
- [ ] Detecção de mãos
- [ ] Processamento de frames em tempo real
- [ ] Integração com IA para análise visual

## 🚀 Como executar
1. Clone o repositório
2. Instale as dependências: `pip install -r requirements.txt`
3. Execute: `python main.py`
